# Monthly-Composite-Country-Risk-Indicator-Germany
## Overview
This project develops a monthly composite indicator designed to track and anticipate German economic activity.

The indicator combines:
- Industrial production
- Retail sales
- Unemployment rate
- Economic Sentiment Indicator (ESI)

using Principal Component Analysis (PCA).

## Objectives
- Build a synthetic indicator of economic activity
- Compare the indicator with quarterly GDP
- Analyze German macroeconomic outlook for 2025–2026
- Develop a short-term nowcasting tool

## Methods
- Time series analysis
- Standardization (z-scores)
- Principal Component Analysis (PCA)
- Linear regression
- Macroeconomic interpretation

## Key Results
- First principal component explains 52% of total variance
- Regression model reaches R² = 0.77
- Indicator successfully captures major German business cycle turning points

## Tools
- R
- Macroeconomic analysis
- Econometrics
- Data visualization
